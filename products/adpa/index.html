Aurreum Data Protection Appliance

Solve data protection problems with simpler, flexible and reliable solutions.

From large corporations that colocate their servers in a data center to small businesses that do not even have a server room, Aurreum provides a turnkey data protection solution for businesses of any size.

Easy Setup

Rack the appliance, power up, and within minutes Aurreum data protection suite will be up and running.

An intuitive Web-based user interface allows effortless configuration and easy operation.

Unlimited Scale-out

Aurreum data protection appliances support cluster configuration with unlimited online scale-out.

The data protection capability can be seamlessly increased by simply dropping in new appliances to join the existing cluster automatically.

Dynamic Load Balance

Not only can a cluster of Aurreum data protection appliances achieve dynamic load balance to provide maximum throughput, but a single appliance with multiple network cards can also achieve the dynamic load balance to avoid the communication bottleneck.

Ransomware protection

Aurreum data protection appliances provide the ultimate solution to ransomware protection.

With the built-in cloud replication and global deduplication, Aurreum data protection appliances can efficiently replicate the backup catalog and data set to a hardened storage volume within the appliance or the object store in any cloud, preventing the attackers from modifying the backup data.

In the event of a ransomware attack, the Aurreum data protection suite utilizes bare metal recovery and instant recovery to quickly restore the backup system and, in turn, restore all the affected production servers to operation.

Flexible solution

Powered by built-in cloud replication and global deduplication, a single rack-mounted data protection appliance meets the data protection needs of most businesses.

For large corporations, a cluster of data protection appliances with unlimited scale-out capability is an ideal solution to protecting their fast-growing business data.

For companies that don't want to own a data protection appliance, Aurreum managed data protection service is their best choice.

For ultra-small businesses that run their business operations primarily through cloud IT services, using a desktop data protection appliance to bring their business data down to the offices gives them greater peace of mind.
