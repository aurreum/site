Aurreum SAP IQ's streaming backup data protection solution makes a stunning debut!

Tue, Jun 4, 2024

Limitations of traditional backup

The traditional backup solution for SAP IQ typically involves the use of specific backup scripts. In this approach, the backup data is first cached locally on the database server and then transferred to the designated backup storage media.

This backup solution has the following limitations, which makes it difficult to improve backup performance and handle large-scale business scenarios:

  • High risk: Manually writing or modifying backup scripts is prone to errors, and maintenance becomes challenging as versions evolve.

  • Consumption of local disk space: The local cache requires additional storage space, especially when dealing with large database sizes.

  • Performance bottlenecks: The process of writing data to the cache before transferring it to the final backup storage media can lead to I/O bottlenecks.

  • Incompatibility with multiple storage media: The scripts are limited to using a single hard disk storage medium and cannot back up to tape libraries, object storage, and other media.

  • Increased management complexity: Managing and maintaining the local disk cache involves complexities such as predefining cache space, cleaning, and addressing cache expiration issues.

The traditional backup approach based on local caching and scripts exhibits significant challenges, particularly when dealing with complex application environments that require scalability, high performance, real-time capabilities, and automation.

Aurreum’s streaming backup solution for SAP IQ

Unlike traditional approaches, Aurreum’s solution eliminates the reliance on scripts backups for SAP IQ. Aurreum has introduced a pioneering streaming backup solution based on the APIs of SAP IQ. In contrast to traditional script-based backups, this solution eliminates the need for intermediate caching. Backup data is transmitted directly to the backup storage media, significantly improving backup efficiency while reducing the utilization of local resources on the database server. This makes the backup of business data in SAP IQ simpler and more efficient.

Key features of Aurreum’s SAP IQ backup solution:

  • Efficient backup without caching: Aurreum’s data backup process does not need local caching, saving local storage space. Compared to competitors, the backup speed is increased by over three times, saving significant time for businesses.

  • Flexible backup strategies: Aurreum offers granular protection through incremental backups and log backups, meeting stringent RPO (Recovery Point Objective) requirements in critical industries. This ensures data integrity and recoverability.

  • Scriptless management: Aurreum eliminates the need for manual script writing and subsequent maintenance through comprehensive automated management based on backup agents. This significantly reduces the operational costs and workload for users.

  • Diverse processing methods: Aurreum supports advanced processing methods for backup data, including distributed deduplication, compression, and encryption. These methods optimize backup data management and storage, enhancing data security and reliability.

  • Multiple storage methods: Aurreum supports directly writing backup data to tape devices or replicating it to off-site locations, meeting industry regulations for long-term retention and offline storage of backup data. This ensures the security and compliance of enterprise data.
